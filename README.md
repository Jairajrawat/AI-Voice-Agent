# My Project

This project combines a Node.js/Express backend with Vocode voice AI capabilities.

## Structure

- `src/` - Express TypeScript backend API
- `vocode-core/` - Vocode voice AI library for building voice-based LLM applications

## Backend Features

- Node.js + Express + TypeScript
- Prisma ORM with PostgreSQL
- JWT authentication
- Zod validation
- ESLint + Prettier

## Voice AI (Vocode)

The `vocode-core` directory contains the Vocode library for building voice-based applications with LLMs.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```
